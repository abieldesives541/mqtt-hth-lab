# mqtt-hth-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常用于展示、测试或作为静态资源引用。仓库内容不针对任何特定域名或网站，仅作为页面文件的集中存放与版本管理使用。

## 目录说明

```
.
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件均为独立页面，可直接在浏览器中打开或部署至静态服务器。
- `assets/` 目录存放页面间共享的样式、脚本及媒体文件，便于维护和复用。

## 页面归档说明

- 所有页面均以独立文件形式存在，互不依赖。
- 页面内容可能包含示例数据、占位文案或测试功能，不构成实际服务或产品。
- 页面文件名通常反映其用途或主题，但具体含义需结合文件内注释理解。
- 归档的页面可能不定期更新或清理，不保证长期保留所有版本。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎提交 Issue 或 Pull Request 以报告问题或建议改进。
- 提交新页面时，请确保文件结构清晰，并尽量在文件头部添加简要注释说明用途。
- 不建议直接依赖本仓库中的文件作为生产环境资源，因其可能随时变更或移除。
